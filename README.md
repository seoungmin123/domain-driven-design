# Domain-Driven Design Study

도메인 주도 설계 기반 이커머스 개발을 위한 학습 프로젝트

## 프로젝트 목표

**최종 목표**: 도메인 주도 설계(DDD)를 적용한 이커머스 시스템 구축

## 브랜치 구성

### java_practice
- 김영한의 자바 중급/고급 개념 학습
- Object, 제네릭, 컬렉션, 람다/스트림 등

### algorithm  
- 백준 알고리즘 문제 풀이
- DFS/BFS, DP, 백트래킹, 시뮬레이션 등

### main
- 최종 DDD 기반 이커머스 프로젝트 (예정)

## 기술 스택

- Java 17+
- Spring Boot (예정)
- JPA/Hibernate (예정)

## 학습 로드맵

1. 자바 기본 개념 학습 (완료)
2. 자바 중급/고급 학습 (진행중)
3. 알고리즘 문제 풀이 (진행중)
4. DDD 개념 학습 (예정)
5. 이커머스 프로젝트 구현 (예정)
